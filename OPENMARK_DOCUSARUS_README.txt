How to use this skeleton (at repo root):

1) Install dependencies:
   npm install

2) Run locally:
   npm run start

3) Build:
   npm run build

4) (Optional) GitHub Pages deploy setup:
   - Ensure in docusaurus.config.js:
     url: 'https://<your_user>.github.io'
     baseUrl: '/<your_repo>/'
