# OpenMark

**OpenMark** turns long ChatGPT conversations into reusable Markdown that anyone can read, improve, and remix.

- 📚 Versioned docs in Git
- 🧩 Simple templates
- 🔁 CI checks for links/build
- 🚀 One-command deploy
