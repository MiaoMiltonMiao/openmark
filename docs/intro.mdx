# Welcome to OpenMark

OpenMark is a **collaborative Markdown knowledge base** powered by Docusaurus.  
Everything starts with **ChatGPT conversations → Markdown Playbooks → Public docs**.

---

## ⚡ Quick Start

1. **Clone this repo**  
   ```bash
   git clone https://github.com/MiaoMiltonMiao/openmark.git
   cd openmark
   npm install
   ```

2. **Local preview**  
   ```bash
   npm run start
   ```
   → open [http://localhost:3000](http://localhost:3000)

3. **Production build**
   - For GitHub Pages:  
     ```bash
     npm run build:gh
     ```
     Output uses `baseUrl=/openmark/`
   - For Vercel:  
     ```bash
     npm run build:vercel
     ```
     Output uses `baseUrl=/`

4. **Deploy**
   - GitHub Pages → auto via Actions (`deploy-pages.yml`)  
   - Vercel → auto from `main` branch

## 🛠️ Authoring Playbooks

All Playbooks live in [playbooks](./playbooks/).  
Each Playbook = one Markdown file, following the [Playbook Template](./template).

### Example
````md
# Playbook title

## 🎯 Goal
Describe what this playbook solves.

## 💡 Mega Prompt
```text
Paste the reusable ChatGPT mega prompt here.
```

## 🧩 Iteration Log
- Try 1 → result
- Try 2 → improved result

## ✅ Final Code / Output
Paste the working snippet / screenshot.
````

---

## 🔄 Iteration & Prompts

- **Mega Prompt on top** → always include the big “starter” instruction.  
- **Log the tries** → short bullet notes of what worked / failed.  
- **Final output** → the polished code / solution at bottom.  

This way, anyone can **reproduce** your workflow, or fork and improve it.
